Project name:   A100 project
Accelerating Connecticut's Startup Careers. A100 is an APPRENTICESHIP PROGRAM for aspiring SOFTWARE developers.
Project name:   White Board project
By applying for the A100 program and getting selected as an Apprentice, you’ll be eligible for our free, 2-month TRAINING series. If you perform well during the training, you’ll be eligible for a paid internship with a local startup. We match all high-performing students with paid internships at growing, profitable startups. During your internship you’ll work on real products for real companies here in Connecticut. We’ll supervise you and make sure that you learn by doing.
